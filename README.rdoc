== BioInteractorNeo4j

Modeling of Biological Interaction Networks (BIN) using Neo4j in Ruby

== Dependencies

* JRuby
* ne04j
* neo4j gem 

==== Install JRuby
The easiest way to install JRuby is by using RVM, see http://rvm.beginrescueend.com. Otherwise check: http://kenai.com/projects/jruby/pages/GettingStarted#Installing_JRuby  

==== Install neo4j
Please refer to: http://www.abhishek-tiwari.com/knowledge-base/installing-neo4j


==== Install the Neo4j Gem

  jruby -S gem install neo4j

If you are using RVM

  gem install neo4j

== Help & Documentation
